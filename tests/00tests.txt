== Test the json generating programs.

The 'reference' output json files are in reference_jsons. I reviewed these output files with my best
effort.

The test samplesheet files are samplesheet*.csv.

The script run01.sh converts these samplesheet*.csv files into samplesheet*.json files.
The script run02.sh runs the make_*_json.py scripts and stores the output files in the directory
test_jsons.
